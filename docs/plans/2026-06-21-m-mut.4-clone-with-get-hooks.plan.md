## 4a — `clone with` (Fork 2 = B: bypasses the constructor)

`obj with { f1 = e1, f2 = e2 }` produces a fresh instance copying `obj`'s fields with the named
ones overridden — **without** running the constructor. No new `Op`: lowers to pushing, in the
class's descriptor field order, either the override expr or `GetField`-from-source, then the
existing `Op::MakeInstance` (which packs fields and runs no ctor body).

- Lexer: `with` keyword (unused as an identifier — verified).
- AST: `Expr::CloneWith { object, fields: Vec<(String, Expr)>, span }`.
- Parser: postfix `with { name = expr, … }` after a primary (trailing comma ok).
- Checker: `object` must be a concrete class `C`; each name a promoted field of `C`; each expr
  assignable to the field's type; result type `C`. Codes `E-WITH-NONCLASS`/`E-WITH-FIELD`/
  `E-WITH-TYPE`.
- Interpreter: clone the `Instance` fields map, override the named ones, fresh `Instance` (no ctor).
- Compiler: resolve `C`'s descriptor idx by name; for each descriptor field emit the override expr
  or `GetLocal(src) + GetField`; `MakeInstance(idx)`.
- Transpiler: `clone(<obj>, ['f1' => e1, …])`.
- Differential `mutation_clone_with_agrees` (override subset, source-unchanged two-binding observe);
  guide `examples/guide/clone-with.phg`.

## 4b — get-hooks (computed/virtual properties)

`class C { …; int area { get => this.w * this.h; } }` — reading `c.area` runs the getter. Replaces
PHP `__get` with a typed, deterministic computed property (capability-preservation map §5).

- AST: a `ClassMember::Field` gains an optional `get` body, or a new `ClassMember::Hook`.
- Checker: `c.area` resolves to the getter's return type; the getter body is checked with `this`.
- Interpreter/VM: a member read of a hook name dispatches the getter (method-on-read).
- Transpiler: PHP property hook `public int $area { get => …; }`.
- Differential + guide `examples/guide/computed-props.phg`.
