# M-mut.1 — Mutable Locals + Reassignment — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or
> superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Add the `mutable` binding modifier and variable reassignment (`x = e;`) to Phorge, byte-identical on
`run ≡ runvm ≡ real PHP`, with immutable-by-default enforcement.

**Architecture:** `mutable` is a *binding* modifier (a `bool` on `Stmt::VarDecl`, never a type). Reassignment
is a new `Stmt::Assign { target: Expr, value: Expr }` whose `target` is an lvalue expression — this slice
accepts only `Expr::Ident` targets (field/index targets land in M-mut.5/6 and extend the *same* statement).
Reassignment reuses the existing `Op::SetLocal` (zero new Op). The interpreter gains an `assign(name, v)` that
overwrites the binding in the scope where `lookup` finds it. `mutable`/immutable is checker-only and **erased**
in PHP output (all PHP locals are mutable).

**Tech Stack:** Rust (edition 2021, std-only, no external crates). `cargo test`, `cargo clippy --all-targets`,
`cargo fmt --check`. PHP oracle via `PHORGE_REQUIRE_PHP=1` (real `php`).

```rust
        Stmt::Assign { target, value, span } => {
            let name = match target {
                Expr::Ident(n, _) => n,
                _ => return Err(self.err_coded(*span,
                        "assignment target must be a simple variable (field/index assignment lands in a later slice)",
                        "E-ASSIGN-TARGET", None)),
            };
            let (ty, mutable) = self.lookup_binding(name).ok_or_else(|| self.err_coded(
                target.span(), &format!("unknown variable `{name}`"), "E-ASSIGN-UNKNOWN", None))?;
            if !mutable {
                return Err(self.err_coded(target.span(),
                    &format!("`{name}` is immutable; declare it `mutable` to reassign"),
                    "E-ASSIGN-IMMUTABLE", Some("add the `mutable` modifier at the declaration")));
            }
            let vty = self.check_expr(value)?;
            if !vty.assignable_to(&ty, &self.subtype_oracle()) {  // match the existing assignability call
                return Err(self.err_coded(value.span(),
                    &format!("cannot assign {vty} to `{name}: {ty}`"), "E-ASSIGN-TYPE", None));
            }
            self.invalidate_narrowing(name);   // S2 smart-cast: reassignment drops any instanceof/optional narrowing
            Ok(())
        }
```

Implement `invalidate_narrowing(name)` against whatever structure stores S2 narrowing (the Explore map points
