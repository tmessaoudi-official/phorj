# Phorj extensions (DEC-273)

Generated by `phg extensions --docs` — do not edit by hand (a test keeps this file in
sync). Run `phg extensions` for the same table plus whether YOUR build carries each one.

The minimal CORE is what the language cannot function without; everything else is an
extension: Rust + JIT like the core, flag-gated for build inclusion only. Importing a
compiled-out extension is `E-EXTENSION-DISABLED`, naming the flag to add. Extensions
keep the `Core.` import root.
